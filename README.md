# Mini Student Task Tracker

A small Next.js + Tailwind app for students to track tasks, assignments, and deadlines.

Quick start:

```bash
# install deps
npm install
# run dev server
npm run dev
```

Features:
- Landing page with Get Started
- Dashboard with summary cards
- Add / Edit / Delete tasks stored in `localStorage`
Design:
- Create mockups in Stitch and add screenshots to `design/` (placeholder)

Notes:
- Tasks persist in browser `localStorage`.
- Filter selection is saved between visits.

Run locally:

```bash
cd mini-student-task-tracker
npm install
npm run dev
# open http://localhost:3000
```
